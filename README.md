## Como usar

Linux
 -
 ###  Instale um ambiente virtual python
 - python3 -m venv venv
  ###  Ative o ambiente virtual
  - source venv/bin/activate
  ### Instale as dependencias
  - pip install -r requirements.txt
  ### Rode o servidor
  - python app.py
  ### Acesse o localhost
  -  No navegador acesse localhost:5000
